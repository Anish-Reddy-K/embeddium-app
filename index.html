About Embeddium

Embeddium is a user-friendly desktop application that democratizes access to vector databases and embeddings. It simplifies complex processes, making powerful embedding tools accessible to non-technical users, small businesses, researchers, and hobbyists alike.

File Input

Support for CSV, JSON, TXT, and XLSX formats. Drag and drop functionality with data preview.

Model Selection

Choose from popular sentence-transformers models. View model descriptions and performance metrics.

Output Options

Select from multiple output formats including .pt, .npy, .hdf5, and .faiss. Customize save location.

Embedding Generation

Real-time progress tracking with cancellation option. View estimated time remaining and processing speed.
